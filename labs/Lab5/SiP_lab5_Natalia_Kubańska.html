Wykorzystując centralne twierdzenie graniczne dla sum pokaż dla rozkładu chi^2 o 2 stopniach swobody jak wygląda suma n niezależnych zmiennych losowych. Pokaż jak zmienia się histogram i czy rzeczywiście rozkład ten dąży do rozkładu normalnego. Sprawdź czy średnia i odchylenie z nowego rozkładu pokrywa się z teoretycznym.

In [6]:
df = 2
num_sim = 10000
n_values = [1, 5, 20, 100]

plt.figure(figsize=(16, 12))

for i, n in enumerate(n_values, 1):
    chi2_samples = np.random.chisquare(df=df, size=(n, num_sim))
    sum_chi2 = np.sum(chi2_samples, axis=0)

    emp_mean = np.mean(sum_chi2)
    emp_std = np.std(sum_chi2)

    theo_mean = n * df
    theo_std = np.sqrt(2 * n * df)

    print(f'n={n}:')
    print(f'Średnia empiryczna: {emp_mean:.2f}, Średnia teoretyczna: {theo_mean:.2f}')
    print(f'Odchylenie empiryczne: {emp_std:.2f}, Odchylenie teoretyczne: {theo_std:.2f}\n')

    plt.subplot(2, 2, i)
    plt.hist(sum_chi2, bins=50, density=True, alpha=0.6, color='salmon', edgecolor='black', label=f'Suma chi^2({df})')

    x_vals = np.linspace(sum_chi2.min(), sum_chi2.max(), 500)
    pdf_vals = norm.pdf(x_vals, loc=theo_mean, scale=theo_std)
    plt.plot(x_vals, pdf_vals, color='blue', lw=2, label=f'N({theo_mean}, {theo_std:.2f})')

    plt.title(f'n={n}, emp_mean={emp_mean:.1f}, emp_std={emp_std:.1f}')
    plt.xlabel('Suma')
    plt.ylabel('Gęstość')
    plt.legend()

plt.tight_layout()

Zadanie 6¶

Winda może przewieźć obciążenie maksymalnie 900 kg. Waga człowieka jest zmienną losową o rozkładzie normalnym N(75,15). Jakie jest prawdopodobieństwo, że przy 13 osobach w windzie, limit wagi nie zostanie przekroczony?

In [7]:
mean_person = 75
std_person = 15
num_people = 13
total_mean = num_people * mean_person
total_std = np.sqrt(num_people) * std_person
limit = 900

z_score = (limit - total_mean) / total_std
probability = norm.cdf(z_score)
print(f'Prawdopodobieństwo, że limit wagi nie zostanie przekroczony: {probability * 100:.1f} %')

Prawdopodobieństwo, że limit wagi nie zostanie przekroczony: 8.3 %

Dla 13 osób prawdopodobieństwo, że limit wagi 900 kg nie zostanie przekroczony wynosi około 8.3%. Oznacza to, że jest wysoce prawdopodobne że limit zostanie przekroczny.
